PHP

¿Que es?

PHP es un lenguaje de scripting de servidor que permite generar páginas web dinámicas. Las páginas PHP pueden contener texto, HTML y bloques de scripts. Cuando un explorador solicita una página PHP, se ejecuta el script PHP en el servidor web y el HTML resultante se muestra en el explorador. Una página PHP debe tener una extensión compatible con PHP. Normalmente, un archivo PHP termina con .php, aunque también existen otras extensiones PHP, como .php4 y .phtml. Sin embargo, .php es la extensión más habitual.
